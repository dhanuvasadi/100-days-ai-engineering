# Day 1 - What is Generative AI?

## 🎯 Learning Objective

Understand what Generative AI is, how it differs from traditional software, and why it is transforming enterprise software development.

---

# What I Learned

## What is Generative AI?

Generative AI is a type of Artificial Intelligence that learns patterns from large amounts of data and generates new content such as text, code, images, or summaries based on a user's prompt.

Unlike traditional software, Generative AI is not programmed with thousands of fixed business rules. Instead, it predicts the most likely response using patterns learned during training.

---

## Traditional Software vs Generative AI

| Traditional Software | Generative AI |
|----------------------|---------------|
| Executes predefined rules | Learns patterns from data |
| Produces deterministic output | Generates new content |
| Requires developers to write business logic | Responds based on prompts and learned patterns |
| Returns existing data | Creates new responses |

---

## Database vs Traditional Software vs Generative AI

### Database

A database stores structured information so that it can be inserted, updated, and queried efficiently.

Examples:
- Oracle
- SQL Server
- PostgreSQL
- Snowflake

---

### Traditional Software

Traditional software executes rules written by developers.

Example:

If Salary > $100,000

Then Apply Tax Rule A

Given the same input, the software produces the same output.

---

### Generative AI

Generative AI learns patterns from large amounts of information and generates new responses based on user prompts.

Instead of retrieving existing answers, it creates responses that are most likely to satisfy the request.

---

# Real-World Enterprise Examples

Generative AI can help organizations:

- Generate SQL queries
- Explain ETL pipelines
- Summarize technical documentation
- Search enterprise knowledge bases
- Generate meeting summaries
- Assist developers with code generation
- Create technical documentation
- Analyze support tickets
